[TIL] list와 tuple의 차이 그리고 dictionary

Author: Sooyoung Jeong Date: 2월 18, 2020  13:40:42 Category: Python TIL

리스트(list)

  • 리스트는 재할당(assignment)이 가능

튜플(tuple)

  • 튜플은 재할당 불가능

딕셔너리(dictionary)

  • key : value 한 쌍으로 이루어진 자료구조
  • 자바스크립트의 객체와 유사

리스트와 딕셔너리의 활용

  • 리스트에서 중복된 요소들을 딕셔너리로 정리
