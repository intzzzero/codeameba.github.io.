하노이의 탑

하노이의 탑은 프랑스 수학자 에두아르 뤼카의 저서를 통해 알려진 게임으로, 다음과 같은 규칙을 갖고 있다.

  • 원반은 한 번에 한 개만 옮길 수 있다.
  • 원반은 막대기 A, B, C 이외의 장소에는 놓을 수 없다.
  • 원반을 옮길 때는 작은 원반 위에 그보다 큰 원반을 쌓을 수 없다.
