Author: Sooyoung Jeong Date: 6월 1, 2019  1:00:42 Category: Exercise

STOP WATCH

Start, Stop 버튼 클릭으로 동작하는 스톱워치 구현
setInterval()과 clearInterval() 메서드가 관건

내용추가
리셋 버튼 추가 1차 시도

  • 초기화 되는 듯 보였지만, start 버튼 누르면 stop 했던 시점부터 다시 시작 됨

2차 시도

  • 스트링 인터폴레이션 사용하여 시간 출력할 때와 동일한 방식으로 적용
  • 제대로 초기화 됨
